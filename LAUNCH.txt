ЗАПУСК СИСТЕМЫ МОНИТОРИНГА

1. ЗАПУСК СЕРВЕРА:
   cd server
   python main.py
   
2. СБОРКА КЛИЕНТА:
   build_client.bat
   
3. УСТАНОВКА КЛИЕНТА В АВТОЗАГРУЗКУ:
   client\install_client.bat
   
4. ДОСТУП К ВЕБ-ИНТЕРФЕЙСУ:
   http://localhost:8000

ПРИМЕЧАНИЯ:
- Сервер должен быть запущен перед установкой клиента
- Клиент работает скрыто в фоне
- Веб-интерфейс обновляется каждые 30 секунд
- Для остановки сервера: Ctrl+C
- Для удаления клиента из автозагрузки: удалить запись из реестра
